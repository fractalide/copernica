# Copernica

Copernica is privacy preserving Information-centric networking protocol designed to operate over UDP and Radio Frequency.

## Getting Started

Install `rustup`.

## Building

`$ rustup run nightly cargo build`

## Running the tests

`$ rustup run nightly cargo test`
